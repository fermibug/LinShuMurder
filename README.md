### 个人猜测

我所认为的事件经过大致是：

受害人在单车棚与王齐莺分别后，为了躲避跑操，从而进入到了三楼厕所等
待。不辛的是张志超也凑巧跑了进来，也许是由于天太黑，张志超以为遇到
了女鬼，在混乱中严重误伤了顾芸，也就是在此时，杨同振与王绪波听到了
凄惨的救命声，直到被惊慌失措的张志超掐住了脖子从而致死，也许当时他
这么做只是想让顾芸停止叫喊。

事后张志超说：“有女鬼”，而且听说还向多个同学提到过 “女鬼”。

> 整个分析建立在王、杨两人的证词之上，即“尖叫”发生时，就是案发时间
